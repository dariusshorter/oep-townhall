# Privacy

The public question form is designed for anonymous submissions.

The application does not intentionally collect or store:

- Name
- Email address
- Authentication identity
- Brand or company
- IP address
- User agent
- Device identifier
- Browser fingerprint
- Location
- Referrer information
- Advertising identifier
- Tracking cookie
- Analytics profile

Participants should not include identifying details in a question unless they choose to do so.
