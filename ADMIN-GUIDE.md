# Administrator Guide

## Daily Review

1. Open `/admin`.
2. Review new questions.
3. Use search and filters to group related questions.
4. Add categories and internal notes.
5. Select questions for presentation.
6. Mark answered questions after the town hall.
7. Archive questions that should no longer appear in active review.

## Presenter Workflow

1. Select questions in `/admin`.
2. Open `/present` on the presentation computer.
3. Use Next and Previous to move through selected questions.
4. Use Mark Answered after a question is addressed.
5. Use Full Screen before screen sharing.

## Status Meanings

- `New`: submitted but not reviewed.
- `Reviewed`: reviewed by an administrator.
- `Selected`: selected for presenter view.
- `Answered`: answered during or after the town hall.
- `Duplicate`: likely duplicate or substantially similar to another question.
- `Archived`: removed from active review.

## Settings

Authorized administrators can change:

- Open or closed submissions
- Town hall name
- Landing prompt
- Success message

These settings are stored in `TownHallSettings` and do not require redeployment.
